# Gestión Financiera - Aplicación Portable v2

## ✅ VERSIÓN CORREGIDA - Pantalla en blanco solucionada

### Descripción
Esta es una aplicación de gestión financiera desarrollada con Electron, React y TypeScript.
Esta versión corrige el problema de pantalla en blanco que tenía la versión anterior.

## 🚀 Cómo ejecutar la aplicación

### Ejecutar directamente
Haz doble clic en el archivo `Gestión Financiera.exe`

## 🎯 Características
- ✅ Interfaz moderna desarrollada con React y Material-UI
- ✅ Gráficos y visualizaciones con Chart.js y Recharts
- ✅ Gestión de gastos e ingresos
- ✅ Categorización de transacciones
- ✅ Filtros y búsquedas avanzadas
- ✅ Soporte para tarjetas de crédito
- ✅ Manejo de cuotas y transacciones futuras
- ✅ Visualización por calendario
- ✅ Múltiples tipos de gráficos (barras, torta, líneas)
- ✅ Almacenamiento local de datos
- ✅ Aplicación completamente portable (no requiere instalación)

## 💻 Requisitos del sistema
- Windows 10 o superior
- No requiere instalación previa de Node.js ni otros requisitos

## 🔧 Correcciones aplicadas
- ✅ Paths relativos corregidos para funcionar en Electron
- ✅ Configuración de webPreferences mejorada
- ✅ Logs de debug añadidos
- ✅ Manejo de errores mejorado
- ✅ Compatible con el puerto correcto del servidor de desarrollo

## 📁 Archivos importantes
- `Gestión Financiera.exe`: Archivo ejecutable principal
- `resources/app.asar`: Contiene el código de la aplicación
- `locales/`: Archivos de localización de Chromium

## 🏗️ Información técnica
- Aplicación construida con Electron 37.2.5
- Frontend desarrollado con React 19.1.0
- UI construida con Material-UI (MUI) 7.2.0
- Empaquetado con @electron/packager
- Base de datos local usando localStorage
